## Installation and Running the app

### Step 1
 ```cd my-app/```
 
 Go to application folder.
 
### Step 2
 ```$ npm install```
 
 Installs project dependencies.

### Step 3
```$ npm run start```

Runs the app in the development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
